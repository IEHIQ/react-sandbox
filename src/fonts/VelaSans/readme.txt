Vela Sans is a modification of Manrope font project. Vela Sans is a free modern grotesque font family. It supports Latin, Cyrillic & Greek languages.

Features
- Semi-condensed, clean, minimal sans-serif font family
- 7 weights
- Extensive language support​​​​​​​
- Comes with desktop and web-font formats
- Packed with OpenType features: Stylistic Sets, Case Sensitive, Auto-Apostrophe, Ligatures, Icon-Ligatures, Tabular Figures

Vela Sans is free for both personal or commercial use.
